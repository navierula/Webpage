Travel Talk
Travel Talk is a social media platform that allows users to create profiles and list their ideal travel destinations.

The backbone of this application was written in Python and the data pertaining to each user was held on a local server at Boston University. In order to allow users easy access to the program, I learned SQL, HTML, and CSS in the span of two weeks to build and ship a simple web application.

If you check it out, please note that oopsies in the demonstration were done on purpose to point out the fact that insertions, deletions, and updates work.
